Главное меню: //Денис
1. Добавить кнопку "добавить транспортное средство"
2. Добавить кнопку "добавить заказ"

Меню машины: //Денис
1. Добавить кнопку вернуться обратно
2. Вывести данные о машине и кнопку начать

Меню грузовика: //Эдгар
1. Добавить кнопку вернуться обратно
2. Слева вывести данные о грузовике. 
3. По середине вывести дневник заказов грузовика. 
4. Справа вывести все заказы в ряд. При нажатии на заказ начинается поездка

Меню Добавить заказ: //Эдгар
1. Добавить кнопку вернуться обратно
2. Название полей
3. Поля заказа
4. Кнопка "Сохранить" >>> При нажатии на кнопку переход на Главное меню

Меню Добавить машину:  //Вместе
1. Добавить кнопку вернуться обратно
2. Название полей
3. Поля тс >>> Если масса тс больше 3500кг, то это грузовик
4. Кнопка "Сохранить" >>> При нажатии на кнопку переход на Главное меню


Игра: //Денис
1. Добавить кнопку вернуться обратно
2. Дорога
3. 2 педали
4. Машина >>> перемещение с помощью педалей
5. Если это грузовик, то при завершении маршрута выполняется заказ >>> Переход в меню грузовика
